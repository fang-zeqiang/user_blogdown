从一次辅导和两次实战经验所获得的宝贵经验，要找到自己在团队中的位置，辩证地看待合作与竞争

为什么会有群面？

  • 本质是筛选人

    1

     e.g.是否能从实习中提炼出东西

  • 群面出的都是简单题目，一般大家都会，实在不会可以选择第二第三个说

  • “无领导小组讨论”是工作场景的常态

  • 角色：leader，协调者timer， 记录，总结，智囊（e.g. 提供方案，后面的人会跟着你走）

  • 要做取舍，要分维度，不能大而全的讨论

特点

  • 群面不确定性高

    e.g. 大的错误比如理解错题目，超时，架空

  • 有同学能点出错误是加分项！

无领导准备

  • 自我介绍，短小精悍，准备亮点和套路？要有意思————”云游20个国家是个在路上的人”

  • 熟悉基本评分规则，比如谁在破题，谁在引导，谁在把握节奏，谁在贡献观点

  • 学会聚焦，构建场景，定义用户群体(e.g. 准中产->大厂上班的员工），以此帮助队伍拉回正轨

  • 找到自己存在的价值，比如身为智囊给团队提示

  • 套路：放弃很多，单拿一线去无限细化缩放落地

流程

  • 切记一些群面最后3min总结陈述不在自由讨论的20min里面，这是timer的责任

  • 既要分配也要抢夺，抢夺比如话语权，标准的树立，也要互帮互助比如纠正错误（微笑着纠正）

  • 竞争，给别人挖坑

    1
2
3

     e.g.   指出对方可行性差比如陈旧，读错题；
        更坏地有延迟几秒再坑人 
        比如 timer怎么不计时了？等等

  • 模型运用能力(比如SWOT,PST)？ 要在框架下聊

    1

    比如四大会出长文，所以有时候不能抠太细，群面挺综合的

评分规则（跳过）

角色定义

  • 记录人/总结人：

    • 有哪些共识，下一步要干什么，有点带节奏的感觉

  • 智囊：

    1. 提供新视角，e.g.建议我们研究周围的人就好，照顾现实问题

    2. 在合适的时间提供建议，比如确定标准，搞出一个新的insight

  • 领导者：

    （容易被挂掉，吃力不讨好）

    1. 帮助引导破题 -> 可能需要有标准，框架
    2. 没有主见hold不住是死穴
    3. leader不是主持人 要带着建议来，别一崩到底
    4. 要把控方向，不能陷入细节！！

  • 计时者：

    1. 还没讨论框架就讨论结果，遇到这种情况能拉回来,即push的能力
    2. 带着时间话题进入讨论，比如”时间还有多少多少我们需要快…”
    3. 需要刷存在感，及时提醒打断超时的人，要有责任心
    4. 比如可以解决矛盾，用一些话术，时间不够啦我这样说可不可以？
    5. 适合不爱说话的人，但是需要不断做小结！！
    6. 可以在短暂的沉默中做个recap，带节奏可以刷存在感加分！！

  • 总结陈述者：

    1. 最容易过，口才好的人适合，有天赋的
    2. 不能紧张说话抖，不能太干啰嗦没有结构化
    3. 不要错误！（总结和讨论过程有出入就是错误点）
    4. 需要有很强的表达能力！有金句最好能升华增加记忆点可以为方案提仙气！
    5. 要逻辑好，可以围绕问题金字塔结构延展

  • 智囊：

    1. hhh这个最懒惰，不累但又能展示自己
    2. 指出别人错误的话术比如“愿望很美好，但ai瓶颈在这…” -> 要把现实摆出来让大家看

    3. 给建设性意见

      e.g. 七个部门分多少给研发？缺少标准 -> 可以和华为类比才20%，一个会计的同学提出的，这时候大家就有了倚仗的讨论标准

    4. 正向、反向思考问题，e.g.加人工，成本怎么办？公司还开吗？

  • 关键点：

    • 要真诚摆事实，提建议要可行！不要太夸大人工智能不要太照顾所有人的情绪，e.g. 双十一的价格歧视，歧视懒人

开放式

  • 先定义问题！ 一般 why 和 what 是最重要的！

  • 美团发了很多补招offer，比如群面问美团的核心竞争力是啥？不是外卖小哥，而是运营团队，商业分析，技术团队的合作

  • 挖掘问题并用一些眼前一亮的insight表达出来 比如 “准中产的理财焦虑”

两难题

why选川普 优点 制造业回流，为了美国 缺点 不理性，对外关系差

why选拜登 优点 理性，和外国搞好关系 缺点 痴呆，没有能力

pick优点规避缺点

排序选择

先定义排序标准（以在互联网加比赛评分为例子） 1. 成果多 2. 项目成熟
